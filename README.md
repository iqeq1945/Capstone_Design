## 핵심기능

### 회원서비스

회원가입,로그인 기능 custom passport 모듈을 활용하여 자체적 인증기능 구현

### 웹소설

소설 작성 및 수정 기능, 검색기능 등 웹소설 플랫폼이 가져야할 기능 구현
Block style의 에디터를 이용

### 결제기능 

Iam port 회사의 api를 이용하여 결제기능을 구현
